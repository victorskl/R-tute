# R-tute

Assorted R Learning

Install R either through [package distribution](https://cran.ms.unimelb.edu.au) or Homebrew:

```
brew info r
brew install r
```

Check version:

```
which R
which Rscript
R --version
```

R Console:

```
$ R
>
> q()
```

Install [**RStudio**](https://www.rstudio.com)
